Primera entrada

Esta es la primera entrada del blog que escribo para la nueva página web.
Cuando digo nueva en realidad me refiero a que la retomé muchos años después de la original
y tiene una filosofía diferente, en esta ocasión en vez de usar las herramientas gratuitas de google
estoy haciendo todo desde cero, todo el código en html y css para únicamente tener una página web estática. La ventaja de esto es que hay mayor seguridad e incluso si la hackearan es mucho más sencillo volver a ponerla en línea.
La otra idea es que sea lo más ligera posible para poder tenerla corriendo con una Raspberry Pi como el servidor, pero eso es el siguiente paso, por el momento yo creo que el host va a ser github, de forma que pueda también practicar esos comandos y únicamente usar mi computadora y subir estas nuevas entradas directamente desde la linea de comandos.
